Financial Modeling Toolkit
Tools for evaluating data‑center and infrastructure investments using modern financial metrics.
Python Pandas NumPy DCF Modeling
Overview

A suite of tools for WACC, NPV, IRR, terminal value, and scenario modeling — tailored to infrastructure and data‑center investing, with an emphasis on clarity and repeatable analysis.

Tech Stack
  • Python + Pandas
  • NumPy for financial math
  • Optional Streamlit UI for interactive dashboards
  • Excel/CSV import and export support
Key Features
  • Discounted cash flow modeling
  • Scenario & sensitivity analysis (e.g., capex, power cost, utilization)
  • Automated charts and visualizations
  • Terminal value and IRR calculators, equity multiple, payback analysis
