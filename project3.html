SpaceX Launch Data Science Project
A full data science workflow analyzing SpaceX Falcon 9 launch data to predict booster landing success, optimize mission reliability, and visualize historical launch performance.
Python Pandas Scikit‑Learn Plotly Dash
Overview

This project explores SpaceX Falcon 9 launch data to understand the factors that influence booster landing success. Using historical launch records, mission metadata, landing outcomes, and payload characteristics, the goal was to build a predictive model and interactive dashboard that helps visualize mission reliability and identify patterns in successful landings.

Tech Stack
  • Python for data cleaning, feature engineering, and modeling
  • Pandas for dataset manipulation and preprocessing
  • Scikit‑Learn for classification models (Logistic Regression, Random Forest, SVM)
  • Plotly Dash for interactive visualizations and mission dashboards
  • Jupyter Notebooks for exploratory data analysis
Data Science Workflow
  • Collected and cleaned SpaceX launch data from public APIs and CSV archives
  • Performed exploratory data analysis to identify correlations between payload mass, orbit type, launch site, and landing success
  • Engineered features such as mission year, booster version, and landing type
  • Trained multiple ML models to predict landing success probability
  • Evaluated models using accuracy, precision, recall, and ROC‑AUC
Key Insights
  • Payload mass and orbit type strongly influence landing success probability
  • Newer booster versions show significantly higher recovery rates
  • Certain launch sites correlate with higher landing reliability
  • Random Forest and Gradient Boosting models performed best in prediction tasks

The final dashboard allows users to explore mission outcomes, filter by orbit or payload mass, and view predicted landing probabilities for future missions.
