Financial Modeling Toolkit

Investment Analysis Tools for Data Centers & Infrastructure

Overview

A collection of financial analysis tools built for evaluating infrastructure and data center investments. Includes calculators for WACC, NPV, IRR, terminal value, equity multiple, and scenario modeling.

Tech Stack

  • Python + Pandas
  • NumPy for financial math
  • Streamlit (optional UI)
  • Excel/CSV import support

Features

  • Discounted cash flow modeling
  • Scenario & sensitivity analysis
  • Automated charts and visualizations
  • Terminal value and IRR calculators
