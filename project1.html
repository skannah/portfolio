AI Roofing Inspection Dashboard
A production‑style SaaS dashboard that turns raw roof images into actionable inspection reports with computer‑vision overlays.
React + Vite Node + Express Roboflow Ant Design
Overview

This app is designed for roofing contractors and insurers who need fast, consistent assessments of hail and wind damage. Users upload drone or smartphone images, then the system runs a trained model and displays detections directly on the image along with key KPIs.

Tech Stack
  • Frontend: React, Vite, Ant Design
  • Backend: Node.js, Express
  • AI: Roboflow‑hosted detection model
  • Infrastructure: GitHub Pages / simple API backend
Key Features
  • Image upload with progress feedback
  • Bounding‑box overlay on predicted damage areas
  • Per‑image and per‑job KPIs (damage counts, severity)
  • Layout ready for multi‑user SaaS plans
