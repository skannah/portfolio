AI Roofing Inspection Dashboard

Full breakdown of the project

Overview

This project analyzes roof images using a Roboflow-trained model, overlays bounding boxes, and displays KPIs in a polished dashboard.

Tech Stack

  • React + Vite
  • Node.js + Express
  • Roboflow Computer Vision Model
  • Ant Design UI

Features

  • Image upload + preview
  • Bounding box overlay
  • Damage classification
  • Dashboard KPIs
