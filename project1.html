AI Roofing Inspection Dashboard
A production‑style SaaS dashboard that turns raw roof images into actionable inspection reports.
React + Vite Node + Express Roboflow Ant Design
Overview

This dashboard allows contractors and insurers to upload roof images, run them through a trained CV model, and view bounding‑box overlays and KPIs in a clean UI.

Tech Stack
  • Frontend: React, Vite, Ant Design
  • Backend: Node.js, Express
  • AI: Roboflow‑hosted detection model
  • Deployment: GitHub Pages (frontend) + API backend
Key Features
  • Image upload with progress and preview
  • Bounding‑box overlay on predicted damage areas
  • Per‑image and per‑job KPIs (damage counts, severity)
  • Layout built for multi‑user SaaS plans
